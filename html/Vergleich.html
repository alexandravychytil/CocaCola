Normal

Cola Original ist das ursprüngliche Rezept und die beliebteste Version von Coca-Cola.
Es hat einen einzigartigen und süssen Geschmack. Die Coca-Cola Original verwendet Zucker als Süssstoff.

Zero

Cola Zero, auch als Diet Coke bekannt, ist eine zuckerfreie Version von Coca-Cola.
Es werden künstliche Süßstoffe wie Aspartam und Acesulfam-Kalium verwendet,
um einen süssen Geschmack ohne zusätzliche Kalorien zu erreichen.

Light

Cola Light ist auch eine low-calorie-Version von Coca-Cola, welche für einen leicht abgänderten Geschmack bekannt ist. Bei der light Version wird auch eine Mischung aus künstlichen Süßstoffen verwendet, jedoch mit einer anderen Zusammensetzung, als die Zero Version. So bietet Cola Light eine weitere kalorienarme Version mit einem süßen abgänderten Geschmack.
